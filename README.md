# Episode Roadmap

This repository contains published snapshots per episode of the RAP vlog [Building a Real RAP Application Step by Step](https://community.sap.com/t5/abap-blog-posts/building-a-real-rap-application-step-by-step-vlog-series-introduction/ba-p/14290896). 

The roadmap below reflects the planned structure of the series. Each episode implements a specific subset of user stories. Episodes are planned to be released on a **weekly basis**.

| Episode | Title | User Stories covered |
| --- | --- | --- |
| 01 | Project introduction + Data model basics | US-01 |
| 02 | Service exposure + First UI (read-only) preview | US-01 |
| 03 | Basic behavior + Search fields (Part 1): Free text + Requester | US-02 (Requester + Free text) |
| 04 | Search fields (Part 2): Status & Priority | US-02 (Status + Priority) |
| 05 | Semantic key & Sequential numbering | US-03 |
| 06 | Mandatory fields + Deadline validation | US-04 + US-05 |
| 07 | Automatic Priority determination + Status icons & visual enhancements | US-06 + US-07 |
| 08 | Status behavior rules | US-08 + US-09 + US-10 |
| 09 | Cancel feature (Part 1) | US-11 + US-12 |
| 10 | Cancel feature (Part 2) | US-13 + US-13b + US-14 |
| 11 | Item popup creation & inline editing | US-18 + US-19 |
| 12 | Product customizing + availability | US-15 + US-16 |
| 13 | Item consistency: prevent duplicates | US-17 + US-20 |
| 14 | Additional rules | US-21 + US-22 |
